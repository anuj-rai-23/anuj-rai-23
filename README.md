## 🚀 Featured Projects

### 🧠 PIB News Retrieval & Summarizer

📌 Problem: Fetch, summarise & organize press releases by Govt. on Infia on Press Information Bureau site (ASP.NET)

⚙️ Solution:
* Built an end-to-end data ingestion and processing pipeline to fetch press releases, parse unstructured HTML, and convert it into structured, machine-readable content

• Designed and implemented automated text summarization workflows to generate concise insights from long-form articles, improving information accessibility and downstream usability

• Developed scalable backend services using Node.js (TypeScript) and Python (FastAPI), enabling modular experimentation and comparison of system performance across architectures

• Created a responsive Vue.js frontend with dynamic grouping, caching strategies, and interactive components to efficiently visualize and explore processed news data

🔗 https://github.com/anuj-rai-23/pib-app
